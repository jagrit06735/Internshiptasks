# Task 2: Basic File Encryption / Decryption

## 📌 Description
This project implements a simple file encryption and decryption system using Python.  
The program allows users to encrypt and decrypt text files using the Caesar Cipher algorithm.

The encrypted content is saved into a new file, and the original content can be restored using the decryption option.

---

## 🎯 Objectives
- Allow user to choose encryption or decryption.
- Take file name as input.
- Encrypt file content and save it as a new file.
- Decrypt the encrypted file back to its original content.
- Handle basic errors (file not found, invalid input).

---

## 🔐 Algorithm Used
**Caesar Cipher**

- Each character is shifted by a fixed number (shift key).
- Encryption shifts forward.
- Decryption shifts backward.
- Only alphabet characters are modified.

# Task 3: N-Queens Problem (Advanced)

## 📌 Description
This project implements the classic N-Queens problem using the Backtracking algorithm in Python.  
The goal is to place N queens on an N×N chessboard such that no two queens threaten each other.

## 🎯 Objectives
- Represent the chessboard as a 2D structure.
- Use backtracking to place queens one by one in safe positions.
- Ensure no two queens share the same row, column, or diagonal.
- Print all possible valid solutions.

## 🧠 Algorithm Used
Backtracking Algorithm:
- Place one queen per row.
- Check column and diagonal safety before placing.
- If conflict occurs, backtrack and try the next position.
